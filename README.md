# PBL Application

## 実行方法

仮想環境を作成していない場合は以下のコマンドで作成

```
python3 -m venv ./venv
```

以下のコマンドで仮想環境に入る。作業する時は常に仮想環境に入ること。

```sh
source ./venv/bin/activate  # 仮想環境に入る
```

仮想環境に入ったら最初は必要なパッケージをインストールする。(以降は不要)

```
pip install -r requirements.txt
```

パッケージがインストールできたら以下のコマンドで実行できる。

```
./run.sh
```
